# 🧮 Calculator | ماشین حساب

A simple calculator with HTML, CSS & JS.  
ماشین‌ حساب ساده با HTML، CSS و JS.

---

## ✅ Features | ویژگی‌ها

- `+ - * / % **`
- Decimal (`.`) & π (`p`)
- Max 10 digits
- Clear & backspace
- Basic error handling

---

## 🚀 How to Use | روش اجرا

1. Clone/download repo  
2. Open `index.html` in browser
